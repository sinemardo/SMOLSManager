# 🍰 Magic Flavors Co

**Magic Flavors Co** es una plataforma digital para el diseño de decoraciones de tartas, bizcochos, picaderas y mesas para eventos.  
Permite a los usuarios crear, personalizar, guardar y compartir diseños visuales de forma sencilla e intuitiva.

---

## 🎯 Objetivo del proyecto

Crear una plataforma:
- Visual e intuitiva (editor gráfico)
- Escalable (web, PWA y móvil)
- Segura (roles y permisos)
- Preparada para monetización (planes y proveedores)

## 👥 Sistema de roles

La plataforma se basa en tres jerarquías principales:

- SuperAdmin  
- Admin (con variantes)  
- Usuario (con variantes)  

La definición completa de roles y permisos se encuentra en el archivo:

**GUIA_ROLES_Y_PERMISOS.md**
