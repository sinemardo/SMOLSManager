# SMOLSManager

**Social Media OnLine Shop Management**

Plataforma para convertir publicaciones de redes sociales en una tienda en linea.

## Estructura

- backend/ - API REST (Express + Prisma + PostgreSQL)
- admin-pwa/ - PWA de gestion (React + Vite)
- web/ - Web publica (Next.js)
- mobile/ - App movil (React Native + Expo)
- docs/mdr/ - 19 ADRs de arquitectura
- .github/ - CI/CD workflows

## Inicio rapido

Backend: cd backend && npm install && npm run dev
PWA: cd admin-pwa && npm install && npm run dev
Web: cd web && npm install && npm run dev
Mobile: cd mobile && npm install && npx expo start
